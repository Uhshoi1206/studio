# **App Name**: Soosan Vina Web

## Core Features:

- Product Catalog: Display a searchable and filterable catalog of Soosan Vina Motor's refrigerated trucks, crane trucks, and semi-trailers.
- News & Updates: Publish news, articles, and updates related to Soosan Vina Motor and the industry.
- SEO Scoring Tool: Provide real-time SEO scoring and feedback within the Decap CMS admin panel, using a tool that checks keyword density and title/description length, and issues warnings
- Product Content Management: Enable content editors to create, update, and manage product information (tieu_de, duong_dan, danh_muc, anh_chinh, bo_suu_tap_anh, gia_ban, thong_so_ky_thuat, seo_tieu_de, seo_mo_ta, tu_khoa_chinh) through Decap CMS.
- News Content Management: Enable content editors to create, update, and manage news articles (tieu_de, duong_dan, tac_gia, ngay_dang, noi_dung, seo_meta) through Decap CMS.
- CDN URL Rewrite: Automatically rewrite local image URLs to the CDN (cdn.soosanmotor.com) if the image exists on the CDN, optimizing image delivery and performance.

## Style Guidelines:

- Primary color: Strong blue (#2962FF) to represent Soosan Vina's dependability. The choice of blue conveys trustworthiness, stability, and technical prowess, aligning well with the automotive industry. Blue is known to inspire confidence and security, essential for attracting potential clients.
- Background color: Very light blue (#E0F7FA), for a clean, modern feel.
- Accent color: Purple (#9FA8DA) to create emphasis and focus user attention to page calls to action.
- Body font: 'PT Sans', a humanist sans-serif, which offers a modern and slightly warm aesthetic, suitable for both headlines and body text.
- Note: currently only Google Fonts are supported.
- Use simple, modern icons to represent product categories and features.
- Implement a clean, responsive layout that is easy to navigate on all devices.
- Use subtle animations to enhance user experience.
